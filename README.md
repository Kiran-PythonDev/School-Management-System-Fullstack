
🏫 School Management System (Full Stack)
A full-stack web application for student data management featuring a ReactJS frontend and Django REST Framework backend.  
🚀 Key Features
Full CRUD Logic: Implemented Create, Read, Update, and Delete operations for student records.  
API Integration: Used Axios for seamless communication between the frontend and the RESTful API.  
State Management: Utilized React Hooks (useState, useEffect) for dynamic data handling.  
Backend Security: Configured CORS middleware for secure cross-origin resource sharing.  
🛠️ Technical Stack
Frontend: ReactJS, JavaScript, Bootstrap.  
Backend: Django, Django REST Framework (DRF).  
Database: MySQL (via XAMPP).  
Tools: Postman (API Testing).
